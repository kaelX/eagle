# eagle(鹰)
---
### 技术栈
  * ruby:  `2.6.1`
  * rails: `5.2.2` 
  * web_service: `puma`
  * database: `pg`
  * api_util: `jbuilder`
  * pagable: `kaminari`
  * httpUtil: `rest-client`
  * cache: `rails-settings-cached redis`
  * encode: `jwt`
  * encrypt/decrypt: `bcrypt and OpenSSL(rsa and aes)`
  * deploy: `capistrano`
  * language: `i18n`
  * test: `Minitest simplecov`
  * data_history: `paper_trail`
